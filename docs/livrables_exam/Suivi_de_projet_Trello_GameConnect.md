# Suivi de projet Trello - GameConnect

## Objectif

Le suivi de projet permet de visualiser l'avancement des taches, les responsabilites et les priorites du projet GameConnect.

## Lien de suivi

Lien Trello fourni dans l'exemple de l'equipe :

https://trello.com/b/QJRSOkjn/agile-scrum-gameconnect

## Organisation recommandee du tableau

| Colonne | Utilite |
| --- | --- |
| Backlog | Idees, besoins et user stories non commencees. |
| A faire | Taches selectionnees pour l'iteration en cours. |
| En cours | Taches en developpement ou redaction. |
| Tests / validation | Taches terminees techniquement mais a verifier. |
| Termine | Taches validees et integrees. |

## Exemples de cartes

| Carte | Type | Priorite |
| --- | --- | --- |
| Creation du compte utilisateur | Fonctionnalite | Haute |
| Connexion JWT | Fonctionnalite | Haute |
| Ajout d'un jeu au profil | Fonctionnalite | Haute |
| Calcul du score de matching | Fonctionnalite | Haute |
| Messagerie entre matchs acceptes | Fonctionnalite | Moyenne |
| Schema MySQL | Base de donnees | Haute |
| Tests frontend Jest | Test | Moyenne |
| Deploiement Docker Compose | Deploiement | Haute |

## Utilisation pour l'examen

Pendant la presentation, ce suivi peut servir a montrer :

- la methode de travail ;
- la repartition des taches ;
- les priorites fonctionnelles ;
- les elements termines et les axes d'amelioration.
